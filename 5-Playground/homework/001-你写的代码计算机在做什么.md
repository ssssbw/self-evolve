# 作业 #001 — 你每天写的代码，计算机到底在做什么？

> 前置阅读：CSAPP 第 1 章「计算机系统漫游」

---

## 题目一（必做）

**请用自己的话描述下面这个过程，越具体越好：**

你写了一行 `int a = 1 + 2;`，从按下键盘到屏幕上显示结果，这之间计算机内部发生了什么？

要求：
- 不要抄书，用你自己理解后的语言写
- 至少覆盖：键盘输入 → CPU → 内存 → 编译 → 执行 → 输出 这条链路
- 字数不限，但每个环节都要写到

提示：第 1 章的「信息 = 位 + 上下文」和「存储器层次结构」是关键线索。

---

## 题目二（必做）

**CSAPP 第 1.4 节讲了「存储器层次结构」：**

```
L0  寄存器
L1  L1 缓存 (SRAM)
L2  L2 缓存 (SRAM)
L3  L3 缓存 (SRAM)
L4  主存 (DRAM)
L5  本地磁盘
L6  远程存储
```

请回答：
1. 为什么需要这么多层？直接用最快的不行吗？
2. 这个层次结构跟你日常用 Flutter 开发有什么关系？（提示：想想图片缓存、数据库查询、懒加载这些概念）
